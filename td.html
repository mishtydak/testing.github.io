BUSINESS CASES

1. Listing of all preferences of users: This case involves creating a system that can collect and display user preferences, such as favorite destinations, activities, or types of accommodations. The system should be able to insert new preferences and update existing ones.

2. Sorting by preference: This case builds on the previous one, where the system sorts user preferences based on their chosen activities. For example, if a user prefers hiking, the system should list hiking-related activities first.

3. Quick visits: In this case, the system recommends places to visit based on their proximity to the user's current location.

4. Listing all the eco-travelling options: This case involves creating a system that lists eco-friendly transportation options, such as walking, biking, or taking public transport.

5. Listing the timings (arrival/departure): Here, the system sorts transportation timings for arrival and departure, ensuring that users can plan their trips efficiently.

6. Shop local: This case promotes local culture by sorting shops alphabetically or by category.
7. Emergency help: In this case, the system provides a list of emergency services, such as hospitals and police stations, to help users in case of an emergency.
